Projeto em Java para o curso de ciência da computação da Faculdade Nove de Julho.

Um software de consultas, cujo nome é Sauni.

Como testar o software:
Para testar o software no seu computador é necessário baixar e colocar o driver mysql-connector-java-8.0.30 na library do arquivo, após isso pegue o script do banco de dados e jogue no seu MySQL, feito isso vá o script SistemaDao e mude as configurações do banco para o que você criou.


Onde ver os scripts:
Para acessar os Scripts basta ir em "SaUni/src/" em ambas as pastas terá os scripts, a pasta "View" onde fica a parte mais visual junto com as interações do usuário e a pasta "Dados" fica a conexão com o banco de dados e as classes onde estão os métodos getters e setters.

Scripts do banco de dados está no arquivo "SCRIPTBD".
